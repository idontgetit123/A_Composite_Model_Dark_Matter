# A Composite Three‑Body Model for Dark Matter
### James Howland (2026)

This repository contains the white paper and technical expansion for a proposed
three‑body composite dark matter candidate consisting of a proton, negaton, and
neutraltron in a stable figure‑8 orbital configuration.
